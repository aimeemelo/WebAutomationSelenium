# Projeto de automação de testes web com Selenium Webdriver + java.

Projeto simples de automação de testes no site : "https://www.qazando.com.br/curso.html"

## Pré-requisitos

É necessário ter o Java, Intellij, navegadores e drivers instalados para executar este projeto.

> Usei as versões `java jdk 1.8` ,'selenium-java 3.141.59', 'Apache Maven 3.8.6' , `cucumber-junit 1.2.5 ` e `ChromeDriver 105.0.51`. Sugiro que você use as mesmas versões ou versões posteriores.
